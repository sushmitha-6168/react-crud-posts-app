# React CRUD Posts App

A modern **CRUD (Create, Read, Update, Delete)** web application built using React. This project demonstrates core frontend development concepts such as routing, API integration, state management, and user interface design.

---

## 🚀 Live Demo

🔗 https://sushmitha-6168.github.io/react-crud-posts-app

---

## 📌 Features

* ➕ Create new posts
* 📄 View posts from API
* ✏️ Edit existing posts
* ❌ Delete posts (UI level)
* 🔄 Client-side routing
* ⚡ Fast and responsive UI
* 🎨 Styled with CSS and interactive effects

---

## 🛠️ Tech Stack

* React (Functional Components & Hooks)
* React Router (Routing)
* JavaScript (ES6+)
* HTML5 & CSS3
* Fetch API
* Git & GitHub

---
